FAQ

Questions, answered.

Is Nova Messages really free?+

Yes. It's free, open-source software under the GPL-3.0 license. No premium tier, no subscriptions, no in-app purchases.

Will it work on my Android?+

Nova targets modern Android versions but stays lightweight enough to run on older devices. If your phone can send SMS, it can run Nova.

Does it sync to the cloud?+

No. Nova has zero network code of its own. SMS / MMS use your carrier the same way any messaging app does — nothing else leaves your phone.

Why isn't it on the Play Store?+

Nova is distributed directly from GitHub Releases as an APK so it stays independent and ad-free. Sideloading takes about 10 seconds.

The app won't open after install — what now?+

On some Androids you must enable Restricted Settings for sideloaded apps, then go to App Info → Storage, tap Clear Cache, Force Stop, and reopen. That fixes it on nearly every device.

Can I contribute?+

Absolutely. Open issues, send PRs, or just star the repo on GitHub. Nova is built in the open.
